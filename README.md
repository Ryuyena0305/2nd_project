# Movie ticket reservation system
미니 프로젝트로 콘솔을 통해 구현한 코드입니다. 
JAVA를 활용하여 "관리자와 고객의 편의성 향상을 위한 프로그램"을 구현하였습니다.


## 👨‍🏫 프로젝트 소개
"최근 몇 년간 비대면 서비스의 필요성이 급격히 증가하면서 고객들은 장소에 구애받지 않고 편리하게 영화 예매를 할 수 있는 온라인 시스템을 제작했습니다.
